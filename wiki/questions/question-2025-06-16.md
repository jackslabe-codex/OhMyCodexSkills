下面 C++ 代码是用欧几里得算法（辗转相除法）求两个正整数的最大公约数，`a` 大于 `b` 还是小于 `b` 都适用。

```cpp
int gcd(int a, int b) {
    while (b) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```
