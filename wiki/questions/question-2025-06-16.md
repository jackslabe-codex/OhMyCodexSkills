## Stem

下面C++代码是用欧几里得算法（辗转相除法）求两个正整数的最大公约数，a 大于 b 还是小于 b 都适用。

```cpp
int gcd(int a, int b) {
    while (b) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

## Answer

true

## Analysis

欧几里得算法（辗转相除法）通过不断取余数来求最大公约数。当a < b时，第一次循环中temp=b, b=a%b=a, a=temp=b，实际上交换了a和b的值，因此无论a大于b还是小于b都适用。
