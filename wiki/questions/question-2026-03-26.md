## 3.1 编程题 1

**试题名称：** 有限不循环小数

**时间限制：** 1.0 s

**内存限制：** 512.0 MB

### 3.1.1 题目描述

若 $\frac{1}{a}$ 可化为一个有限的，不循环的小数，则称 $a$ 为终止数。

请你求出在 $L$ 到 $R$ 中终止数的数量。

### 3.1.2 输入格式

输入一行，包含两个整数 $L, R$。

### 3.1.3 输出格式

输出一行，包含一个整数，表示 $L$ 到 $R$ 中终止数的数量。

### 3.1.4 样例

#### 3.1.4.1 输入样例

```text
2 11
```

#### 3.1.4.2 输出样例

```text
5
```

### 3.1.5 样例解释

在 $[2,11]$ 终止数有 2、4、5、8、10。

### 3.1.6 数据范围

保证 $1 \le L \le R \le 10^6$。

### 3.1.7 参考程序

```cpp
#include <iostream>

using namespace std;

int main() {
    int l, r, ans = 0;
    cin >> l >> r;
    for(int i = l; i <= r; i++) {
        int t = i;
        while(t && t % 2 == 0)
            t /= 2;
        while(t && t % 5 == 0)
            t /= 5;
        if(t == 1)
            ans++;
    }
    cout << ans;
    return 0;
}
```
