## Stem

现在用如下代码来计算 x^n （n个x相乘），其时间复杂度为（ ）。

```cpp
double quick_power(double x, unsigned n) {
    if (n == 0) return 1;
    if (n == 1) return x;
    return quick_power(x, n / 2) * quick_power(x, n / 2) * ((n & 1) ? x : 1);
}
```

A. O(n)

B. O(n²)

C. O(logn)

D. O(n logn)

## Answer

A

## Analysis

该快速幂实现存在重复计算问题：`quick_power(x, n/2)` 被调用了两次。设T(n)为计算x^n的时间复杂度，则有递推式 T(n) = 2*T(n/2) + O(1)。根据主定理或递归树分析，该递推式的解为 T(n) = O(n)。
