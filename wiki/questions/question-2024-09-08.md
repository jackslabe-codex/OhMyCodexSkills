现在用如下代码来计算 `x^n`（`n` 个 `x` 相乘），其时间复杂度为（ ）。

```cpp
double quick_power(double x, unsigned n) {
    if (n == 0) return 1;
    if (n == 1) return x;
    return quick_power(x, n / 2) * quick_power(x, n / 2) * ((n & 1) ? x : 1);
}
```

A. `O(n)`
B. `O(n^2)`
C. `O(log n)`
D. `O(n log n)`
