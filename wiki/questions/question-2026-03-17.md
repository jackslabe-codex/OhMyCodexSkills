## Stem

若数组 a 已按升序排列，则下面代码可以正确实现在 a 中查找第一个大于等于 x 的元素的位置。

```cpp
int lowerBound(vector<int>& a,int x){
    int l=0, r=a.size();
    while(l < r) {
        int mid = (l + r) / 2;
        if( a[mid] >= x) r = mid;
        else l = mid + 1;
    }
    return l;
}
```

## Answer

true

## Analysis

该代码实现了二分查找（lower_bound），用于在升序数组中查找第一个大于等于x的元素位置。算法逻辑正确，当a[mid] >= x时右边界左移，否则左边界右移，最终返回第一个满足条件的位置。
