## Stem

下面的C++代码使用数组模拟整数加法，可以处理超出大整数范围的加法运算。横线处应填入代码是（ ）。

```cpp
vector<int> operator +(vector<int> a, vector<int> b)
{
    vector<int> c;
    int t = 0;
    for(int i = 0; i < a.size() || i < b.size(); i ++ )
    {
        if(i < a.size()) t = t + a[i];
        if(i < b.size()) t = t + b[i];
        ____________________
    }
    if(t) c.push_back(t);
    return c;
}
```

A. c.push_back(t % 10), t = t % 10;
B. c.push_back(t / 10), t = t % 10;
C. c.push_back(t / 10), t = t / 10;
D. c.push_back(t % 10), t = t / 10;

## Answer

D

## Analysis

该代码模拟大整数加法。变量 `t` 存储当前位的和及进位。根据加法规则，当前位应取 `t % 10`（个位数），进位应为 `t / 10`（十位数及以上）。因此应先将 `t % 10` 压入结果向量，再更新 `t = t / 10` 以便下一位计算。
