## Stem

第15题 小杨编写了一个如下的高精度乘法函数，则横线上应填写的代码为（ ）。

```cpp
vector<int> multiply(vector<int>& a, vector<int>& b) {
    int m = a.size(), n = b.size();
    vector<int> c(m + n, 0);

    // 逐位相乘，逆序存储
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            c[i + j] += a[i] * b[j];
        }
    }

    // 处理进位
    int carry = 0;
    for (int k = 0; k < c.size(); ++k) {
        ________________________________     // 在此处填入代码
        c[k] = temp % 10;
        carry = temp / 10;
    }

    while (c.size() > 1 && c.back() == 0)
        c.pop_back();
    return c;
}
```

A. `int temp = c[k];`
B. `int temp = c[k] + carry;`
C. `int temp = c[k] - carry;`
D. `int temp = c[k] * carry;`

## Answer

B

## Analysis

高精度乘法处理进位时，每一位的最终值等于当前位累加的乘积值加上低位的进位 carry。因此 `temp` 应初始化为 `c[k] + carry`，然后取模得到当前位数字，整除得到新的进位。
