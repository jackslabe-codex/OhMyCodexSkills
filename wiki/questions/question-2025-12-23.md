## Stem

以下 fib 函数计算第 n 项斐波那契数（fib(0)=0, fib(1)=1），其时间复杂度为 O(n)。

```cpp
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}
```

## Answer

false

## Analysis

该递归实现的时间复杂度为 O(2^n)（指数级），而非 O(n)。因为存在大量重复计算（如 fib(5) 需要计算多次 fib(3)）。要达到 O(n) 复杂度需要使用动态规划或记忆化搜索。
