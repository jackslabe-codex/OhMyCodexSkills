## Stem

下面代码尝试在有序数组中查找第一个大于等于 x 的元素位置。如果没有大于等于 x 的元素，返回 `arr.size()`。以下说法正确的是（ ）。

```cpp
int lower_bound(vector<int>& arr, int x) {
    int l = 0, r = arr.size();
    while(l < r) {
        int mid = l + (r - l) / 2;
        if(arr[mid] >= x) r = mid;
        else l = mid + 1;
    }
    return l;
}
```

A. 上述代码逻辑正确
B. 上述代码逻辑错误，while 循环条件应该用 l <= r
C. 上述代码逻辑错误，mid 计算错误
D. 上述代码逻辑错误，边界条件不对

## Answer

A

## Analysis

该代码实现了标准的 lower_bound 二分查找逻辑。搜索区间为 [l, r)，其中 r 初始化为 arr.size()（即数组末尾的下一个位置），循环条件为 l < r。当 arr[mid] >= x 时，说明答案可能在 mid 或其左侧，令 r = mid；否则令 l = mid + 1。最终返回 l，若所有元素都小于 x 则返回 arr.size()，逻辑完全正确。
