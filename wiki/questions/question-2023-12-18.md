## Stem

以下 C++ 代码能以递归方式实现斐波那契数列，该数列第 1、2 项为 1，以后各项均是前两项之和。( )

```cpp
int Fibo(int N)
{
    if (N == 1 || N == 2)
        return 1;
    else
    {
        int m = fiboA(N - 1);
        int n = fiboB(N - 2);
        return m + n;
    }
}
```

## Answer

false

## Analysis

该代码存在两个明显的错误：1. 函数名定义为 `Fibo` 但内部调用的是 `fiboA` 和 `fiboB`，函数名不匹配会导致编译错误；2. 即使修正函数名，这种未使用记忆化的递归实现斐波那契数列的时间复杂度为指数级 $O(2^N)$，效率极低。
