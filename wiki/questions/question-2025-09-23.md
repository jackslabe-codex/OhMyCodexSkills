## Stem

下面代码采用分治算法求解标准3柱汉诺塔问题，时间复杂度为 $O(2^n)$。

```cpp
void move(vector<int> &src, vector<int> &tar) {
    int pan = src.back();
    src.pop_back();
    tar.push_back(pan);
}

void dfs(int n, vector<int> &src, vector<int> &buf, vector<int> &tar) {
    if (n == 1) {
        move(src, tar);
        return;
    }
    dfs(n - 1, src, tar, buf);
    move(src, tar);
    dfs(n - 1, buf, src, tar);
}

void solveHanota(vector<int> &A, vector<int> &B, vector<int> &C) {
    int n = A.size();
    dfs(n, A, B, C);
}
```

## Answer

false

## Analysis

标准3柱汉诺塔问题的递推关系为$T(n)=2T(n-1)+1$，解得时间复杂度为$O(2^n)$。代码采用分治策略：先将n-1个盘子从源柱移到辅助柱，再将第n个盘子从源柱移到目标柱，最后将n-1个盘子从辅助柱移到目标柱。
