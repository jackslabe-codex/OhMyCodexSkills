## Stem

假设函数 gcd() 函数能正确求两个正整数的最大公约数，则下面的 lcm(a, b) 函数能正确找到两个正整数 a 和 b 的最小公倍数。

```cpp
int lcm(int a, int b) {
    return a / gcd(a, b) * b;
}
```

## Answer

true

## Analysis

最小公倍数公式为 $lcm(a,b) = \frac{a \times b}{gcd(a,b)}$。代码中先进行除法 `a / gcd(a, b)` 再乘以 `b`，可以避免整数溢出（在 $a \times b$ 超出 int 范围时），逻辑正确。
