## Stem

假设函数 gcd() 函数能正确求两个正整数的最大公约数，则下面的 lcm() 函数能求相应两数的最小公倍数。

```cpp
int lcm(int a, int b) {
    return a * b / gcd(a, b);
}
```

## Answer

true

## Analysis

该函数在计算a*b时可能发生整数溢出（特别是当a和b都接近INT_MAX时）。正确的做法应该是先进行除法运算：a / gcd(a, b) * b。
