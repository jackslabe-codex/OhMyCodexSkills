## Stem

下面代码实现了欧拉（线性）筛，横线处应填写（ ）。

```cpp
vector<int> euler_sieve(int n) {
    vector<bool> is_composite(n + 1, false);
    vector<int> primes;

    for (int i = 2; i <= n; i++) {
        if (!is_composite[i])
            primes.push_back(i);

        for (int j = 0; __________________________ && (long long)i * primes[j] <= n; j++) {
            is_composite[i * primes[j]] = true;

            if (i % primes[j] == 0)
                break;
        }
    }
    return primes;
}
```

A. `j <= n`
B. `j < sqrt(n)`
C. `j < primes.size()`
D. `j < i`

## Answer

C

## Analysis

欧拉筛的内层循环遍历已找到的质数列表primes，因此循环条件应为j < primes.size()（或j < (int)primes.size()）。当i%primes[j]==0时break保证每个合数只被最小质因子筛一次。
