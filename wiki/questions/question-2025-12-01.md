对如下定义的循环单链表，横线处填写（ ）。

// 循环单链表的结点
struct Node {
    int data;        // 数据域
    Node* next;      // 指针域
    Node(int d) : data(d), next(nullptr) {}
};

// 创建一个只有一个结点的循环单链表
Node* createList(int value) {
    Node* head = new Node(value);
    head->next = head;
    return head;
}

// 在循环单链表尾部插入新结点
void insertTail(Node* head, int value) {
    Node* p = head;
    while (p->next != head) {
        p = p->next;
    }
    Node* node = new Node(value);
    node->next = head;
    p->next = node;
}

## Answer

C

## Analysis

循环单链表的特点是尾节点的next指针指向头节点，因此遍历时不能使用nullptr作为终止条件（选项A、B、D均会导致死循环或逻辑错误）。正确的做法是使用do-while结构，先输出当前节点数据，再移动指针，直到回到头节点时停止（选项C）。
