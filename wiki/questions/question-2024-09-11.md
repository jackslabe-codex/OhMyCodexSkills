## Stem

第 11 题
现在有 `n` 个人要过河，每只船最多载 2 人，船的承重为 100kg。下列代码中，数组 `weight` 中保存有 `n` 个人的体重（单位为 kg），已经按从小到大排好序，代码输出过河所需要的船的数目，采用的思想为（ ）。

```cpp
int i, j;
int count = 0;
for (i = 0, j = n - 1; i < j; j--) {
    if (weight[i] + weight[j] <= 100) {
        i++;
    }
    count++;
}
printf("过河的船数：%d\n", count);
```

A. 枚举算法
B. 贪心算法
C. 迭代算法
D. 递归算法
