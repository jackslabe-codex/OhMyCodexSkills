## Stem

现在有 $n$ 个人要过河，每只船最多载2人，船的承重为100kg。下列代码中，数组 `weight` 中保存有 $n$ 个人的体重（单位为kg），已经按从小到大排好序，代码输出过河所需要的船的数目，采用的思想为（ ）。

```cpp
int i, j;
int count = 0;
for (i = 0, j = n - 1; i < j; j--) {
    if (weight[i] + weight[j] <= 100) {
        i++;
    }
    count++;
}
printf("过河的船数：%d\n", count);
```

A. 枚举算法

B. 贪心算法

C. 迭代算法

D. 递归算法

## Answer

B

## Analysis

该代码采用双指针策略：指针 `i` 从最轻的人开始，指针 `j` 从最重的人开始。每次尝试让最重的人（`j`）与最轻的人（`i`）同船（如果体重和不超过100kg），否则最重的人单独乘船。这种“每次都做局部最优选择（尽可能让两人同船以节省船数）”的策略符合贪心算法的特征。
