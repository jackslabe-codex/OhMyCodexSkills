## Stem

函数 puzzle 定义如下，则调用 puzzle(7) 程序会无限递归。

```cpp
int puzzle(int n) {
    if (n == 1) return 1;
    if (n % 2 == 0) return puzzle(n / 2);
    return puzzle(3 * n + 1);
}
```

## Answer

false

## Analysis

调用puzzle(7)：7是奇数且不等于1，执行puzzle(3*7+1)=puzzle(22)；22是偶数，执行puzzle(11)；11是奇数，执行puzzle(34)；34是偶数，执行puzzle(17)；17是奇数，执行puzzle(52)...这个过程不会无限递归，最终会到达1（这是考拉兹猜想/Collatz conjecture，对于7会经过16步到达1）。
