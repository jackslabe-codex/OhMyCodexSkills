## Stem

下面代码实现了欧几里得算法。下面有关说法，错误的是（ ）。

```cpp
int gcd1(int a, int b) {
    return b == 0 ? a : gcd1(b, a % b);
}

int gcd2(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

A. gcd1() 实现为递归方式。
B. gcd2() 实现为迭代方式。
C. 当a较大时，gcd1() 实现会多次调用自身，需要较多额外的辅助空间。
D. 当a较大时，gcd1() 的实现比 gcd2() 执行效率更高。

## Answer

D

## Analysis

gcd1()使用递归实现欧几里得算法（辗转相除法），gcd2()使用迭代实现。两者时间复杂度相同（均为O(log min(a,b))），但递归实现需要额外的栈空间来保存调用帧，当a较大时递归深度可能较深，空间开销更大。因此说gcd1()执行效率更高是错误的（两者效率相同或递归略慢于迭代）。
