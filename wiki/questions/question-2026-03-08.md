## Stem

在升序数组中查找第一个大于等于 x 的位置，下面循环中横线应填（ ）。

```cpp
int lowerBound(const vector<int>& a, int x){
    int l=0, r=a.size();
    while(l<r){
        int mid = l + (r - l)/2;
        if(a[mid] >= x) _____________;
        else l = mid + 1;
    }
    return l;
}
```

## Answer

A

## Analysis

lower_bound查找第一个>=x的位置。当a[mid]>=x时，说明目标位置在左半部分（含mid），应令r=mid；否则l=mid+1。这是二分查找找下界的标准模板。
