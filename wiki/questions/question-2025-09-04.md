## Stem

函数 isPerfectNumber 判断一个正整数是否为完全数（该数是否即等于它的真因子之和），则横线上应填写（ ）。一个正整数 n 的真因子包括所有小于 n 的正因子，如28的真因子为1, 2, 4, 7, 14。

```cpp
bool isPerfectNumber(int n) {
    if(n <= 1) return false;
    int sum = 1;
    for(int i = 2; ______; i++) {
        if(n % i == 0) {
            sum += i;
            if(i != n/i) sum += n/i;
        }
    }
    return sum == n;
}
```

A. i <= n
B. i*i <= n
C. i <= n/2
D. i < n

## Answer

B

## Analysis

判断完全数时，需要枚举真因子。对于因子i，其对应的另一个因子为n/i。为了避免重复枚举和越界，只需枚举到sqrt(n)，即i*i <= n。同时注意处理i与n/i相等的情况（如36=6×6）。
