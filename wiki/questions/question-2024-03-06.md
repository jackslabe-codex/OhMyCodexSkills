下面的代码片段用于计算斐波那契数列。该代码的时间复杂度是（ ）？

```cpp
int fibonacci(int n) {
    if (n <= 1) {
        return n;
    } else {
        return fibonacci(n - 1) + fibonacci(n - 2);
    }
}
```

A. `O(1)`
B. `O(n)`
C. `O(2^n)`
D. `O(n^2)`
