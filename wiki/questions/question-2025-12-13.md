## Stem

下面给出了阶乘计算的两种方式。以下说法正确的是（ ）。

```cpp
int factorial1(int n) {
    if (n <= 1) return 1;
    return n * factorial1(n - 1);
}

int factorial2(int n) {
    int acc = 1;
    while (n > 1) {
        acc = n * acc;
        n = n - 1;
    }
    return acc;
}
```

A. 上面两种实现方式的时间复杂度相同，都为 $O(n)$
B. 上面两种实现方式的空间复杂度相同，都为 $O(n)$
C. 上面两种实现方式的空间复杂度相同，都为 $O(1)$
D. 函数 factorial1() 的时间复杂度为 $O(2^n)$，函数 factorial2() 的时间复杂度为 $O(n)$

## Answer

A

## Analysis

factorial1 采用递归实现，时间复杂度为 O(n)，空间复杂度为 O(n)（递归调用栈深度）。factorial2 采用迭代实现，时间复杂度为 O(n)，空间复杂度为 O(1)（仅使用常数个变量）。因此两种实现时间复杂度相同但空间复杂度不同，只有选项 C 关于空间复杂度的描述是错误的（说两者相同且为 O(1) 不对）。
