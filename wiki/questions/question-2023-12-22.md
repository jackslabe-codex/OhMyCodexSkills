## Stem

下面的 C++ 代码能实现十进制正整数 N 转换为八进制并输出。( )

```cpp
char s[10];
int main()
{
    int N;
    cin >> N;
    string rst = "";
    while (N != 0)
    {
        s[0] = N % 8 + '0';
        rst += string(s);
        N /= 8;
    }
    cout << rst << endl;
    return 0;
}
```

## Answer

false

## Analysis

该代码逻辑错误：它将每次计算得到的余数字符串 `s` 直接拼接到了结果字符串 `rst` 的末尾（第 10 行 `rst += string(s)`），而没有将新计算的位放在高位（即字符串前面）。这会导致输出的八进制数的各位顺序颠倒（例如输入 9 会输出 "01" 而不是 "1"，或更复杂的数字会完全错位）。正确的做法应该是将 `s` 拼接到 `rst` 的前面，或者最后反转字符串。
