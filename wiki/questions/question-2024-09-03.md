## Stem

第 3 题
对下面两个函数，说法错误的是（ ）。

```cpp
int sumA(int n) {
    int res = 0;
    for (int i = 1; i <= n; i++) {
        res += i;
    }
    return res;
}

int sumB(int n) {
    if (n == 1)
        return 1;
    int res = n + sumB(n - 1);
    return res;
}
```

A. `sumA` 体现了迭代的思想。
B. `sumB` 采用的是递归方式。
C. `sumB` 函数比 `sumA` 的时间效率更高。
D. 两个函数的实现的功能相同。

## Answer

C
