## Stem

对下面两个函数，说法错误的是（ ）。

```cpp
int sumA(int n) {
    int res = 0;
    for (int i = 1; i <= n; i++) {
        res += i;
    }
    return res;
}

int sumB(int n) {
    if (n == 1)
        return 1;
    int res = n + sumB(n - 1);
    return res;
}
```

A. sumA体现了迭代的思想。

B. SumB采用的是递归方式。

C. SumB函数比SumA的时间效率更高。

D. 两个函数的实现的功能相同。

## Answer

C

## Analysis

sumA使用for循环实现迭代求和，时间复杂度为O(n)；sumB使用递归实现求和，时间复杂度也为O(n)。两者时间效率相同。由于递归存在函数调用栈的开销，实际运行中迭代通常略快于递归。因此说SumB的时间效率更高是错误的。
