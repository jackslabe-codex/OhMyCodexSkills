## Stem

函数 linearSieve 实现线性筛法(欧拉筛)，横线处应填入（ ）。

```cpp
vector<int> linearSieve(int n) {
    vector<bool> is_prime(n+1, true);
    vector<int> primes;
    for(int i = 2; i <= n; i++) {
        if(is_prime[i]) primes.push_back(i);
        for(int p : primes) {
            if(p * i > n) break;
            is_prime[p * i] = false;
            if(________) break;
        }
    }
    return primes;
}
```

A. i % p == 0
B. p % i == 0
C. i == p
D. i * p == n

## Answer

A

## Analysis

线性筛法（欧拉筛）的关键在于保证每个合数只被其最小质因子筛一次。当i % p == 0时，说明p是i的最小质因子，此时必须break，否则后续更大的质因子乘以i会产生重复标记（且最小质因子不再是该大质因子）。
