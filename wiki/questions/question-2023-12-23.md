## Stem

对数组 `int arr[] = {2, 6, 3, 5, 4, 8, 1, 0, 9, 10}` 执行 `sort(arr, arr+10)` ，则执行后 arr 中的数据调整为 `{0, 1, 2, 3, 4, 5, 6, 8, 9, 10}` 。( )

## Answer

true

## Analysis

C++ 标准库中的 `sort` 函数默认按升序排列。对数组 `{2, 6, 3, 5, 4, 8, 1, 0, 9, 10}` 执行升序排序后，结果确实为 `{0, 1, 2, 3, 4, 5, 6, 8, 9, 10}`。
