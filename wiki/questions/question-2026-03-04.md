## Stem

对如下代码实现的欧几里得算法（辗转相除法），执行 gcd(48, 18) 得到的调用序列为（ ）。

```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}
```

A. `gcd(48,18) -> gcd(18,12) -> gcd(12,6) -> gcd(6,0)`
B. `gcd(48,18) -> gcd(30,18) -> gcd(12,18)`
C. `gcd(48,18) -> gcd(18,30) -> gcd(30,6)`
D. `gcd(48,18) -> gcd(12,18) -> gcd(6,12)`

## Answer

A

## Analysis

gcd(48,18)：48%18=12，调用gcd(18,12)；gcd(18,12)：18%12=6，调用gcd(12,6)；gcd(12,6)：12%6=0，调用gcd(6,0)；gcd(6,0)返回6。故调用序列为gcd(48,18)->gcd(18,12)->gcd(12,6)->gcd(6,0)。
