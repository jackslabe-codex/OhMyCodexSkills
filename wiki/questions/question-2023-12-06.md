## Stem

有关下面C++代码说法正确的是（ ）。

```cpp
int rc;
int foo(int x, int y)
{
    int r;
    if(y == 0)
        r = x;
    else{
        r = foo(y, x % y);
        rc++;
    }
    return r;
}
```

A. 如果 x 小于10，rc 值也不会超过20
B. foo 可能无限递归
C. foo 可以求出 x 和 y 的最大公共质因子
D. foo 能够求出 x 和 y 的最小公倍数

## Answer

A

## Analysis

该函数实现的是辗转相除法（欧几里得算法）求最大公约数(GCD)，且由于每次递归调用时 y 变为 x%y，若初始 x,y 为正整数则 y 严格递减，必然终止不会无限递归。当 x,y 互质时 GCD=1（非质数），故只能求最大公约数而非最大公共质因子。rc 作为全局变量记录递归次数，x<10 时 rc 可能超过20（如 foo(9,1) 会递归多次）。
