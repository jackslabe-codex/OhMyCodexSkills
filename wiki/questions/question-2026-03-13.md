## Stem

有 n 位同学的成绩已经从小到大排好序，现在对它执行下面这段以第一个元素为 pivot 的快速排序，请问此次排序的时间复杂度是（ ）。

```cpp
void quicksort(vector<int>& a, int l, int r) {
    if (l >= r) return;
    int pivot = a[l];
    int i = l, j = r;
    while (i < j) {
        while (i < j && a[j] >= pivot) j--;
        while (i < j && a[i] <= pivot) i++;
        if (i < j) swap(a[i], a[j]);
    }
    swap(a[l], a[i]);
    quicksort(a, l, i - 1);
    quicksort(a, i + 1, r);
}
```

A. O(n)
B. O(nlogn)
C. O(n²)
D. O(logn)

## Answer

C

## Analysis

数组已经有序且选取第一个元素作为pivot，每次划分只能确定一个元素的位置（最坏情况），导致快速排序退化为O(n²)的时间复杂度。
