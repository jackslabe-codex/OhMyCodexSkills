## Stem

有如下函数 fun ，则 fun(20, 12) 的返回值为（ ）。

```cpp
int fun(int a, int b) {
    if (a % b == 0)
        return b;
    else
        return fun(b, a % b);
}
```

A. 20

B. 12

C. 4

D. 2

## Answer

C

## Analysis

该函数实现的是辗转相除法（欧几里得算法）求最大公约数。计算过程为：fun(20, 12) -> fun(12, 8) -> fun(8, 4)，此时8%4==0，返回4。
