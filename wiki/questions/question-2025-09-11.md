## Stem

第11题 下述C++代码实现了快速排序算法，下面说法错误的是（ ）。

```cpp
int partition(vector<int>& arr, int low, int high) {
    int i = low, j = high;
    int pivot = arr[low];                 // 以首元素为基准
    while (i < j) {
        while (i < j && arr[j] >= pivot) j--;   //从右往左查找
        while (i < j && arr[i] <= pivot) i++;   //从左往右查找
        if (i < j) swap(arr[i], arr[j]);
    }
    swap(arr[i], arr[low]);
    return i;
}

void quickSort(vector<int>& arr, int low, int high) {
    if (low >= high) return;
    int p = partition(arr, low, high);
    quickSort(arr, low, p - 1);
    quickSort(arr, p + 1, high);
}
```

A. 快速排序之所以叫“快速”，是因为它在平均情况下运行速度较快，常数小、就地排序，实践中通常比归并排序更高效。
B. 在平均情况下，划分的递归层数为log n，每层中的总循环数为n，总时间为O(nlogn)。
C. 在最差情况下，每轮划分操作都将长度为n的数组划分为长度为0和n-1的两个子数组，此时递归层数达到n，每层中的循环数为n，总时间为O(n²)。
D. 划分函数partition中“从右往左查找”与“从左往右查找”的顺序可以交换。

## Answer

D

## Analysis

在以首元素为基准的快速排序划分函数中，必须先从右向左查找小于基准的元素，再从左向右查找大于基准的元素。如果交换这两个查找顺序，当数组本身有序或逆序时，会导致指针越界或错误的交换，破坏算法正确性。因此“从右往左查找”与“从左往右查找”的顺序不可随意交换。
