## Stem

第8题 对下面两个函数，说法错误的是（ ）。

```cpp
int factorialA(int n) {
    if (n <= 1) return 1;
    return n * factorialA(n-1);
}
int factorialB(int n) {
    if (n <= 1) return 1;
    int res = 1;
    for(int i=2; i<=n; i++)
        res *= i;
}
```

A. 两个函数的实现的功能相同。
B. 两个函数的时间复杂度均为O(n)。
C. factorialA采用递归方式。
D. factorialB采用递归方式。

## Answer

D

## Analysis

factorialA采用递归方式实现阶乘，factorialB采用迭代（循环）方式实现阶乘。两者功能相同，时间复杂度均为O(n)。选项D称factorialB采用递归方式是错误的，它实际采用的是迭代方式。
