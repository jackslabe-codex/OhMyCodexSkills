## Stem

3.1 编程题 1

题目描述

试题名称：平均分配

时间限制：1.0 s

内存限制：512.0 MB

小 A 有 $2n$ 件物品，小 B 和小 C 想从小 A 手上买走这些物品。对于第 $i$ 件物品，小 B 会以 $b_i$ 的价格购买，而小 C 会以 $c_i$ 的价格购买。为了平均分配这 $2n$ 件物品，小 A 决定小 B 和小 C 各自只能买走恰好 $n$ 件物品。你能帮小 A 求出他卖出这 $2n$ 件物品所能获得的最大收入吗？

输入格式

第一行，一个正整数 $n$。

第二行，$2n$ 个整数 $b_1,b_2,\ldots,b_{2n}$。

第三行，$2n$ 个整数 $c_1,c_2,\ldots,c_{2n}$。

输出格式

一行，一个整数，表示答案。

样例

输入样例 1：

```text
3
1 3 5 6 8 10
2 4 6 7 9 11
```

输出样例 1：

```text
36
```

输入样例 2：

```text
2
6 7 9 9
1 2 10 12
```

输出样例 2：

```text
35
```

数据范围

对于部分测试点，保证数据规模较小。

对于所有测试点，保证 $1 \le n \le 10^5$，$0 \le b_i,c_i \le 10^9$。

参考程序

```cpp
#include <bits/stdc++.h>

using namespace std;

const int N = 2e5 + 5;

int n;
long long b[N], c[N], d[N];
long long ans;

int main() {
    scanf("%d", &n);
    assert(1 <= n && n <= 1e5);
    for (int i = 1; i <= 2 * n; i++)
        scanf("%lld", &b[i]), assert(0 <= b[i] && b[i] <= 1e9);
    for (int i = 1; i <= 2 * n; i++)
        scanf("%lld", &c[i]), assert(0 <= c[i] && c[i] <= 1e9);
    for (int i = 1; i <= 2 * n; i++) {
        ans += b[i];
        d[i] = c[i] - b[i];
    }
    sort(d + 1, d + 2 * n + 1);
    for (int i = n + 1; i <= 2 * n; i++)
        ans += d[i];
    printf("%lld\n", ans);
    return 0;
}
```
