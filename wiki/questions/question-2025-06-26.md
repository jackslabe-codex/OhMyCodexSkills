## 奖品兑换

**时间限制**: 1.0 s

**内存限制**: 512.0 MB

### 题目描述

班主任给上课专心听讲、认真完成作业的同学们分别发放了若干张课堂优秀券和作业优秀券。同学们可以使用这两种券找班主任兑换奖品。具体来说，可以使用 $a$ 张课堂优秀券和 $b$ 张作业优秀券兑换一份奖品，或者使用 $b$ 张课堂优秀券和 $a$ 张作业优秀券兑换一份奖品。

现在小 A 有 $n$ 张课堂优秀券和 $m$ 张作业优秀券，他最多能兑换多少份奖品呢？

### 输入格式

第一行，两个正整数 $n, m$，分别表示小 A 持有的课堂优秀券和作业优秀券的数量。

第二行，两个正整数 $a, b$，表示兑换一份奖品所需的两种券的数量。

### 输出格式

输出共一行，一个整数，表示最多能兑换的奖品份数。

### 参考程序

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

typedef long long LL;
LL n, m, a, b;

// 检查是否可以兑换 k 份奖品
bool check(LL k) {
    // 方案1: 使用k份第一种方式(a,b)，剩余的用第二种方式(b,a)尽可能多换
    // 第一种方式消耗: k*a 张课堂券, k*b 张作业券
    // 剩余课堂券: n - k*a, 剩余作业券: m - k*b
    // 第二种方式最多能换: min((n-k*a)/b, (m-k*b)/a)
    // 总数需要 >= k
    if (k * a <= n && k * b <= m) {
        LL remain_n = n - k * a;
        LL remain_m = m - k * b;
        if (remain_n / b + remain_m / a >= k) return true;
    }

    // 方案2: 使用k份第二种方式(b,a)，剩余的用第一种方式(a,b)尽可能多换
    if (k * b <= n && k * a <= m) {
        LL remain_n = n - k * b;
        LL remain_m = m - k * a;
        if (remain_n / a + remain_m / b >= k) return true;
    }

    return false;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    cin >> n >> m >> a >> b;

    LL l = 0, r = 2e9; // 上界设为足够大
    while (l < r) {
        LL mid = (l + r + 1) >> 1;
        if (check(mid)) {
            l = mid;
        } else {
            r = mid - 1;
        }
    }

    cout << l << endl;
    return 0;
}
```
