## Stem

第10题 给定一个n x n的矩阵matrix，矩阵的每一行和每一列都按升序排列。函数countLE返回矩阵中第k小的元素，则两处横线上应分别填写（ ）。

```cpp
// 统计矩阵中 <= x 的元素个数：从左下角开始
int countLE(const vector<vector<int>>& matrix, int x) {
    int n = (int)matrix.size();
    int i = n - 1, j = 0, cnt = 0;
    while (i >= 0 && j < n) {
        if (matrix[i][j] <= x) {
            cnt += i + 1;
            ++j;
        }
        else {
            --i;
        }
    }
    return cnt;
}

int kthSmallest(vector<vector<int>>& matrix, int k) {
    int n = (int)matrix.size();
    int lo = matrix[0][0];
    int hi = matrix[n - 1][n - 1];
    while (lo < hi) {
        int mid = lo + (hi - lo) / 2;
        if (countLE(matrix, mid) >= k) {
            ________________    // 在此处填入代码
        } else {
            ________________    // 在此处填入代码
        }
    }
    return lo;
}
```

```
lo = mid + 1;
```

## Answer

C

## Analysis

该函数使用二分查找在有序矩阵中寻找第k小的元素。`countLE(matrix, mid)`返回矩阵中不超过mid的元素个数。当该值大于等于k时，说明第k小元素在左半区间（含mid），应执行`hi = mid`；当小于k时，说明第k小元素在右半区间（不含mid），应执行`lo = mid + 1`。这是典型的二分查找下界模板。
