## Stem

函数 sieve 实现埃拉托斯特尼筛法(埃氏筛)，横线处应填入（ ）。

```cpp
vector<bool> sieve(int n) {
    vector<bool> is_prime(n+1, true);
    is_prime[0] = is_prime[1] = false;
    for(int i = 2; i <= n; i++) {
        if(is_prime[i]) {
            for(int j = ______; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    return is_prime;
}
```

A. i
B. i+1
C. i*2
D. i*i

## Answer

D

## Analysis

埃拉托斯特尼筛法中，对于素数i，从i*i开始标记其倍数为合数。因为小于i*i的合数已经被更小的素因子筛过了（如i=5时，10已被2筛过，15已被3筛过，20已被2筛过，所以从25开始）。虽然从i*2开始也可以，但i*i效率更高。
