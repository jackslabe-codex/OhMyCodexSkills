## Stem

两块长方形土地的长宽分别为 24 和 36 米，要将它们分成正方形的小块，使得正方形的尺寸尽可能大。小杨采用如下的辗转相除函数 gcd(24, 36) 来求正方形分块的边长，则函数 gcd 调用顺序为（ ）。

```cpp
int gcd(int a, int b) {
    int big = a > b ? a : b;
    int small = a < b ? a : b;
    if (big % small == 0) {
        return small;
    }
    return gcd(small, big % small);
}
```

A. gcd(24, 36)、gcd(24, 12)、gcd(12, 0)

B. gcd(24, 36)、gcd(12, 24)、gcd(0, 12)

C. gcd(24, 36)、gcd(24, 12)

D. gcd(24, 36)、gcd(12, 24)

## Answer

C

## Analysis

根据代码逻辑，gcd(24,36) 中 big=36, small=24，调用 gcd(24, 12)；接着 gcd(24,12) 中 big=24, small=12，且 24%12=0，返回 12。因此调用顺序为 gcd(24,36) → gcd(12,24) → 返回（或视为 gcd(0,12) 终止）。选项 B 符合辗转相除的参数传递顺序。
