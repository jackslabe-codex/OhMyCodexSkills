## Stem

第4题 用以下辗转相除法（欧几里得算法）求gcd(84, 60)的步骤中，第二步计算的数是（ ）。

```cpp
int gcd(int a, int b) {
    int big = a > b ? a : b;
    int small = a < b ? a : b;
    if (big % small == 0) {
        return small;
    }
    return gcd(small, big % small);
}
```

A. 84和60
B. 60和24
C. 24和12
D. 12和0

## Answer

B

## Analysis

使用辗转相除法求gcd(84,60)：第一步计算84%60=24，递归调用gcd(60,24)；第二步计算60%24=12，递归调用gcd(24,12)；第三步计算24%12=0，返回12。因此第二步计算的数是60和24。
