用以下辗转相除法（欧几里得算法）求 `gcd(84, 60)` 的步骤中，第二步计算的数是（ ）。

```cpp
int gcd(int a, int b) {
    int big = a > b ? a : b;
    int small = a < b ? a : b;
    if (big % small == 0) {
        return small;
    }
    return gcd(small, big % small);
}
```

A. 84和60

B. 60和24

C. 24和12

D. 12和0
