## Stem

下面函数用“哑结点”统一处理删除单向链表中的头结点与中间结点。横线处应填（ ）。

```cpp
struct Node{
    int val;
    Node* next;
    Node(int v):val(v),next(nullptr){}
};

Node* eraseAll(Node* head, int x){
    Node dummy(0);
    dummy.next = head;
    Node* cur = &dummy;
    while(cur->next){
        if(cur->next->val == x){
            Node* del = cur->next;
            ______________________
            delete del;
        }else cur = cur->next;
    }
    return dummy.next;
}
```

A. `cur = cur->next;`
B. `cur->next = del->next;`
C. `del->next = cur->next;`
D. `cur->next = nullptr;`

## Answer

B

## Analysis

使用哑结点统一处理删除操作时，cur指向待删节点的前驱，del指向待删节点。删除del需要将cur->next指向del的下一个节点，即cur->next = del->next，然后释放del。
