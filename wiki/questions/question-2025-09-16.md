## Stem

基于下面定义的函数，通过判断 `isDivisibleBy9(n) == isDigitSumDivisibleBy9(n)` 代码可验算如果一个数能被9整除，则它的各位数字之和能被9整除。

```cpp
bool isDivisibleBy9(int n) {
    return n % 9 == 0;
}

bool isDigitSumDivisibleBy9(int n) {
    int sum = 0;
    string numStr = to_string(n);
    for (char c : numStr) {
        sum += (c - '0');
    }
    return sum % 9 == 0;
}
```

## Answer

true

## Analysis

根据数学性质，一个数能被9整除当且仅当其各位数字之和能被9整除。代码中isDivisibleBy9判断n%9==0，isDigitSumDivisibleBy9计算各位数字之和并判断sum%9==0，两者逻辑等价。
