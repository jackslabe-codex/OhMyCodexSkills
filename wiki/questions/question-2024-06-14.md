## Stem

给定如下函数：

```cpp
int fun(int n) {
    if (n == 1) return 1;
    if (n == 2) return 2;
    return fun(n - 2) - fun(n - 1);
}
```

则当 n=7 时，函数返回值为（ ）。

A. 0
B. 1
C. 21
D. -11

## Answer

D

## Analysis

递推计算：fun(1)=1, fun(2)=2, fun(3)=fun(1)-fun(2)=1-2=-1, fun(4)=fun(2)-fun(3)=2-(-1)=3, fun(5)=fun(3)-fun(4)=-1-3=-4, fun(6)=fun(4)-fun(5)=3-(-4)=7, fun(7)=fun(5)-fun(6)=-4-7=-11。
