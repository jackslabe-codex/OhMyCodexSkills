## Stem

下面是根据欧几里得算法编写的函数，它计算的是a与b的（ ）。

```cpp
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

A. 最小公倍数
B. 最大公共质因子
C. 最大公约数
D. 最小公共质因子

## Answer

C

## Analysis

该代码实现了欧几里得算法（辗转相除法），通过不断用较大数除以较小数取余，直到余数为0，最后的除数即为最大公约数（GCD）。
