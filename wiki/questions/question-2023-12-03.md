## Stem

阅读下面的C++代码，执行后其输出是( )。

```cpp
int stepCount = 0;
int fracA(int N)
{
    stepCount += 1;
    cout << stepCount << "->";
    int rtn = 1;
    for (int i = 1; i <= N; i++)
        rtn *= i;
    return rtn;
}
int fracB(int N)
{
    stepCount += 1;
    cout << stepCount << "->";
    if (N == 1)
        return 1;
    return N * fracB(N - 1);
}
int main()
{
    cout << fracA(5);
    cout << "<===>";
    cout << fracB(5);
    return 0;
}
```

A. 1->120<===>2->120
B. 1->120<===>1->120
C. 1->120<===>1->2->3->4->5->120
D. 1->120<===>2->3->4->5->6->120

## Answer

D

## Analysis

fracA(5) 执行时 stepCount 从0开始累加5次变为5，输出 "1->120"；fracB(5) 执行时 stepCount 先加1变为6，输出 "6->"，然后 N=5≠1 返回 5*fracB(4)。fracB(4) 使 stepCount 变为7输出 "7->" 返回 4*fracB(3)... 最终返回 5!=120 并输出 "120"。因此完整输出为 1->120<===>2->3->4->5->6->120。
