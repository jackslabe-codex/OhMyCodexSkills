## Stem

以下代码计算两个正整数的最大公约数(GCD)，横线上应填写（ ）。

```cpp
int gcd0(int a, int b) {
    if (a < b) {
        swap(a, b);
    }
    while(b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return ______;
}
```

A. b
B. a
C. temp
D. a * b

## Answer

B

## Analysis

这是辗转相除法（欧几里得算法）求GCD的实现。循环结束后，b变为0，此时a存储的就是最大公约数，因此返回a。
