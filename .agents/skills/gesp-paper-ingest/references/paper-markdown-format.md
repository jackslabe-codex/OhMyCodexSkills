# GESP 真题 PDF 转 Markdown 格式需求

```text
请将 1 份 GESP 真题 PDF 转成 1 个 Markdown 文件，严格按以下格式输出：

- 文件扩展名必须是 `.md`
- 正文第一行格式：`GESP C++ 五级 YYYY年M月真题`
- 必须包含 3 个一级分段标题：
  - `## 单选题`
  - `## 判断题`
  - `## 编程题`

单选题格式：
- 单选题标题下必须先给答案表，格式固定为：
  `| 题号 | 1 | 2 | 3 | ... |`
  `|---|---|---|---|---|`
  `| 答案 | A | B | C | ... |`
- 每题标题必须是：`### 第 N 题`
- 题干放在题号下方
- 选项必须写成：
  - `A. ...`
  - `B. ...`
  - `C. ...`
  - `D. ...`

判断题格式：
- 判断题标题下必须先给答案表，格式固定为：
  `| 题号 | 1 | 2 | 3 | ... |`
  `|---|---|---|---|---|`
  `| 答案 | T | F | T | ... |`
- 每题标题必须是：`### 第 N 题`
- 题干放在题号下方

编程题格式：
- 每题标题必须是：
  - `### 3.1 编程题 1`
  - `### 3.2 编程题 2`
- 每题内部必须按顺序包含：
  - `#### 题目描述`
  - `#### 输入格式`
  - `#### 输出格式`
  - `#### 样例`
  - `#### 样例解释`
  - `#### 数据范围`
  - `#### 参考程序`
- 所有代码必须放在 fenced code block 中，例如 ```cpp

禁止事项：
- 不要输出 HTML
- 不要输出图片链接
- 不要省略题号
- 不要省略单选题答案表
- 不要省略判断题答案表
- 不要把代码变成普通段落
- 不要改成别的标题名称
```
